# Event Driven Orders

Software Architecture Lab.

Proyecto orientado al aprendizaje de arquitecturas modernas utilizando:

- Java 21
- Spring Boot
- Apache Kafka
- RabbitMQ
- Docker
- Kubernetes
- AWS

## Objetivos

- Comprender Event Driven Architecture.
- Dominar Apache Kafka.
- Implementar microservicios desacoplados.
- Aplicar TDD y buenas prácticas.
- Construir un proyecto con calidad empresarial.
